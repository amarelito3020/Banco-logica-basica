# 💳 Simulador de Banco em Java

Este projeto é uma aplicação simples feita em Java para simular operações bancárias básicas. O foco é praticar conceitos de **Programação Orientada a Objetos (POO)**, **encapsulamento**, **construtores**, **métodos** e **tratamento de exceções**.

## 🚀 Funcionalidades

- Abertura de conta (Conta Corrente ou Conta Poupança)
- Bonificação automática ao abrir conta:
  - Conta Corrente: R$150
  - Conta Poupança: R$300
- Depósitos e saques
- Pagamento de mensalidade
- Encerramento de conta com validação
- Validação de tipo de conta
- Uso de `IllegalArgumentException` para garantir entrada válida

## 🛠️ Tecnologias

- Java 17+
- IntelliJ IDEA (opcional)
- Git

## 🧠 Conceitos Aplicados

- Programação Orientada a Objetos
- Encapsulamento de atributos
- Construtores personalizados
- Controle de fluxo (if/else)
- Manipulação de exceções
